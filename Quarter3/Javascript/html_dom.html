HTML DOM & Forms

1a.) DOM is the document object model. One is created any time a browser loads a web page. It is constructed as a tree of objects, which include elements like < head > and < body >.

2a.) The < html > element is the parent node of the < body > element.

2b.) The child nodes of the < body > element include < a > and < h1 >.

2c.) The sibling of the < body > element is the < head > element.
