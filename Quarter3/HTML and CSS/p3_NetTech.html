Computer Netwerken

Das Internet besteht aus Netzwerken unterschiedlicher administrativer Verwaltung, die zusammengeschaltet sind. Darunter sind hauptsaechlich Providernetzwerke, an die die Rechner der Kunden eines Internetproviders angeschlossen sind, Firmennetzwerke (Intranets), die die Computer einer Firma verbinden, sowie Universitaets- und Forschungsnetzwerke.

Das Internet

In den Nutzdaten des Internetprotokolls werden abhaengig vom verwendeten Dienst immer noch Protokolle hoeherer Schichten (wie TCP oder UDP) uebertragen, so wie ein ISO-Container im Gueterverkehr Postpakete beinhalten kann, in denen wiederum Gueter eingepackt sind. Die meisten Webseiten benutzen, aufbauend auf TCP, das Hypertext Transfer Protocol (HTTP) und das Hypertext Transfer Protocol Secure (HTTPS) fuer verschluesselte Seiten. E-Mails benutzen das Simple Mail Transfer Protocol (SMTP), ebenfalls aufbauend auf TCP, das DNS wird dagegen weitgehend mittels UDP abgewickelt.

Das WWW

Das WWW ist ein ueber das Internet abrufbares System von elektronischen Hypertext-Dokumenten, sogenannten Webseiten. Sie sind durch Hyperlinks untereinander verknuepft und werden im Internet ueber die Protokolle HTTP oder HTTPS uebertragen. Die Webseiten enthalten meist Texte, oft mit Bildern und grafischen Elementen illustriert. Haeufig sind auch Videos, Tondokumente und Musikstuecke eingebettet.

Web 2.0

Web 2.0 ist ein Schlagwort, das fuer eine Reihe interaktiver und kollaborativer Elemente des Internets, speziell des World Wide Webs, verwendet wird. Dabei konsumiert der Nutzer nicht nur den Inhalt, er stellt als Prosument selbst Inhalt zur Verfuegung. Der Begriff postuliert in Anlehnung an die Versionsnummern von Softwareprodukten eine neue Generation des Webs und grenzt diese von frueheren Nutzungsarten ab. Die Verwendung des Begriffs nimmt jedoch zugunsten des Begriffs Social Media ab.

Das Mobile Internet

Mobiles Internet bezeichnet die Bereitstellung einer Internetverbindung oder von Teilen davon (z. B. World Wide Web) auf Mobilgeraeten wie insbesondere Laptops, Handys und Tablets.
